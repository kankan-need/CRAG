双模列尾系统由列尾设备和附属设备沮成，设备包括列尾机车台和列尾主机。

# 5.2系统功能

双模列尾系统功能应满以下要求：

a) 列尾主机能采集列车尾部主风管风压。  
b) 列尾机车台与列尾主机建立连接关系。  
c ) 列尾机车台向列尾主机动查询列车尾部风压。  
d) 列尾机车台按规定的时间间隔动向列尾主机查询列车尾部风压。超过规定时间没有收到列尾信息时，列尾机车台发出通信失效告警。  
e) 列尾主机检测到列车尾部主风管风压低于设定值时，动向列尾机车台报警。  
f ) 列尾主机电池电量不时，动向列尾机车台报警。  
g) 列尾主机标识列车尾部。  
h) 列尾机车台控制列尾主机辅助排风。  
i) 列尾机车台与列尾主机销号。  
j） 列尾机车台与列尾主机断连。  
k) 列尾机车台和列尾主机向RMS系统传送在途运状态信息。  
1) 列尾机车台通过“数据Ⅰ”接口连接CIR卫星定位单元公用接口获取位置和时间等信息自动校准本机时间,与卫星定位信息时间误差不于 $3 \mathrm { ~ s ~ }$   
m) 出库检测台对列尾机车台进检测。  
n) 列尾主机检测台对列尾主机进检测。  
0) 处于连接状态下，列尾机车台和列尾主机能进辅助排风制动、电池电量不报警、主风管风压不正常报警和风压动态显示。  
p) 处于连接状态下，如列尾主机时钟与列尾机车台时钟误差大于5s，则列尾主机动按列尾机车台时钟校准本机时钟。  
q） 列尾机车台和列尾主机断电重启后自动进未连接状态；列尾机车台和列尾主机自动复位则恢复原连接状态；如通过列尾控制盒手动复位，列尾机车台重启后恢复原连接状态。  
r) 列尾机车台、列尾主机未存储对IP时，能通过列尾ID号、机车号查询对IP并存储。  
s) 列尾机车台、列尾主机具备 $4 0 0 ~ \mathrm { M H z }$ 数字和双模两种作模式。

# 5.5试验方法

# 5.5.1常温试验

5.5.1.1 400MHz数字通信模块 按GB/T32659—2016进试验。

# 5.5.1.2 GSM-R模块

按YD/T12152006进试验。

# 5.5.2 功能试验

在400MHz、GSM-R环境下，列尾机车台和列尾主机应通过带安全编码的双模列尾标准（协议)模拟器进功能试验和互联互通测试。

# 5.5.3 外观检查

用量具测量列尾主机外形尺寸、重量，列尾主机标志，列尾主机标志发光强度按照TB/T2353-

2018进测试;量具测量列尾机车台主机(含减振架)和列尾控制盒外形尺寸。

# 5.5.4低温试验

按GB/T2423.12008中试验Ad进。列尾主机温度为 $- 3 0 ~ \mathrm { { ^ { \circ } C } }$ ，列尾机车台温度为 $- 2 5 \ \mathrm { ~ ‰ ~ }$ ,持续时间 $^ { 2 \mathrm { ~ h ~ } }$ ,放置时间终了，在保持低温状态下对装置通电，并进性能检测。恢复后，在正常室温下重新进性能检测。

# 5.5.5高温试验

按GB/T2423.2—2008中试验Bd进，采然通风；装置通电后，放在试验箱内，在不小于$0 . 5 \mathrm { ~ h ~ }$ 内将箱温从正常试验环境温度 $2 5 ~ \mathrm { ^ { \circ } C } \pm 1 0 ~ \mathrm { ^ { \circ } C }$ 逐渐升至列尾主机温度为 $+ 5 5 ~ \mathrm { { ‰ } }$ ，列尾机车台主机温度为 $+ 6 5 \ \mathrm { ~ \textdegree C }$ ，列尾控制盒温度为 $+ 4 5 \ \mathrm { ~ \textdegree C }$ ，待温度稳定后，将装置放置 $^ { 2 \mathrm { ~ h ~ } }$ ，然后，在已升高的温度下进性能检测。恢复后，在常温下重新进性能检测。

# 5.5.6湿热试验

按GB/T2423.4的规定进。除性能检测外，不应对被试装置通电。

具体条件为：

a 温度为 $+ 5 5 \ \mathrm { { ~ ‰ ~ } }$ 和 $+ 2 5 \mathrm { ~ \mathscr ~ { ~ C ~ } ~ }$ ;  
b) 周期数为2（「乎吸效应）；  
时间为 $2 \times 2 4 \mathrm { ~ h ~ }$ ;  
d)a） 性能检测；  
b） 外观检查。

# 5.5.7高低温交热试验

列尾主机按GB/T2423.222012中7.2规定进试验，保持时间 $^ \textrm { \scriptsize 3 h }$ ，循不5次。

# 5.5.8电磁兼容试验

按GB/T24338.4—2018进行。

# 5.5.9 振动和冲击试验

按GB/T215632018中1类B级要求进，加速度例系数取7.83。

# 5.5.10 密封防护试验

按GB/T42082017进试验。

# 5.5.11 由跌落试验

按GB/T2423.7进试验，跌落次数为2次。

# 6.3技术要求

# 6.3.1环境要求

# 6.3.1.1 正常工作环境

# 6.3.1.1.1海拔

不超过 $3 ~ 0 0 0 ~ \mathrm { m }$

# 6.3.1.1.2环境温度

可控列尾主机环境温度为 $- 3 0 \mathrm { ~ \mathcal { C } \sim } + 5 5 \mathrm { ~ \mathcal { C } ~ }$ ,可控列尾控制盒环境温度为 $- 2 5 \mathrm { ~ } \mathrm { { ~ C ~ } } { \sim } + 4 5 \mathrm { ~ } \mathrm { { ~ C ~ } }$ 。

# 6.3.1.1.3相对湿度

温度保持 $3 0 ~ \mathrm { { ^ { \circ } C } }$ 不变时,相对湿度不于 $9 5 \% \mathrm { R H }$

# 6.3.1.1.4外壳防护

设备外壳防护等级应符合GB/T42082017的规定，可控列尾主机达到IP53的等级要求。可控列尾控制盒达到IP52的等级要求。

# 6.3.1.1.5电磁兼容

符合GB/T24338.42018表3和表6的规定。

# 6.3.1.1.6振动和冲击

符合GB/T215632018中1类B级规定。

# 6.3.1.1.7 自由跌落

可控列尾主机跌落度 $1 \ 0 0 0 \ \mathrm { m m }$ ，跌落向正向(作向)后正常作。

# 6.3.1.2特殊工作环培

当使条件与6.3.！.1有差异时，使本件的各可在设备采购时另商定。

# 6.3.2 通信方式

可控列尾系统采GSM-R络下的CSDF进数据传输。

# 6.3.3主要技术特性

# 6.3.3.1可控列尾主机

6.3.3.1.1可控列尾主机标志的显示应符合下列要求：  
a）昼间采红相间的标志,标志尺寸不于 $1 5 0 \ \mathrm { m m } \times 1 0 0 \ \mathrm { m m }$ ;  
b）夜间采动闪光标志，发红光强度不于 $5 ~ \mathrm { c d }$   
6.3.3.1.2 天线的技术要求应符合TB/T18751987中便携式电台螺旋鞭状天线的有关规定。6.3.3.1.3 能适应列车制动主管定压 $6 0 0 ~ \mathrm { k P a }$ 的要求，风压反馈最允许误差为 $5 \ \mathrm { k P a }$   
6.3.3.1.4 软管的性能和胶料技术要求应符合TB/T602014的相关规定。  
6.3.3.1.5 采用可控式电磁排风阀进同步制动和辅助排风，同步制动减压速率应满足车辆制动机$1 0 ~ \mathrm { k P a / s } \sim 4 0 ~ \mathrm { k P a / s }$ 的要求，且回波的压差应小于 $5 \ \mathrm { k P a }$ 。  
6.3.3.1.6 同步制动时，尾部实际减压量误差不于 $1 0 \ \mathrm { k P a }$ ，且不于机车减压量。  
6.3.3.1.7质量不大于 $1 2 ~ \mathrm { k g }$ (不含电池)。  
6.3.3.1.8外形尺寸（宽 $\times$ 高 $\times$ 厚)不于 $4 0 0 \ \mathrm { m m } \times 5 0 0 \ \mathrm { m m } \times 3 0 0 \ \mathrm { m m } \cdot$ （不含风管）。  
6.3.3.1.9 可控列尾主机应牢固地安装在列车最后车辆末端的车钩上，其风管与列车尾部主风管可靠连接。安装装置应适合各种车钩，并有锁闭装置。

# 附录A（规范性）双模列尾系统附属设备

# A.1 车载线通信设备出入库检测台

# A.1.1设备组成

出入库检测台包括工控机 $、 4 0 0 ~ \mathrm { M H z }$ 数字通信模块、卫星定位单元、UPS、打印机、天馈线、体化机柜和CIR各部检测组件等，实现对双模列尾机车台和CIR的出库检测。

# A.1.2 设备主要功能

出库检测台设备主要功能应满下列要求：

a）具有检测双模列尾机车台和CIR的功能；  
b) 具有双模列尾机车台和CIR故障报警功能；  
c) 具有查询、显和打印检测结果的功能；  
d) 具有存储检测效据能，存储时间不于 $3 0 \mathrm { ~ d ~ }$ ;  
e) 具有连接RMS系统实现化管理的功能。

# A.1.3 检测方式

出库检测台检测式应满下列要求：

a） 机车出库时，由维护员上车对双模列尾机车注检测，检测结果保存在地的出库检测台； 注行  
b) 双模列尾机车台预存配置（GSM-R或 $4 0 0 ~ \mathrm { M H z }$ 数字通信模块控制盒数量等配置信息和库检台IP地址，出库检测台将本机IP地址的后两字节作为出库检测台设备编号（16进制，字节在前)。

# A.1.4启动检测

检测员在双模列尾机车台选择或输出库检测台IP地址后，双模列尾机车台进出库检测状态，同时启动数据测试和检。

# A.1.5 数据测试

双模列尾机车台按照预存的通信模块配置信息，同时进以下数据测试，数据测试应满下列要求：

a） 双模列尾机车台配置了GSM-R模块时，如GSM-R注册正常，则向出库检测台发送GSM-R数据测试命令，收到出库检测台应答后判为GSM-R数据正常， $1 0 \mathrm { ~ s ~ }$ 内未收到出入库检测台应答进重发，重发2次后仍未收到出库检测台应答判为GSM-R数据故障；如GSM-R未注册，直接判为GSM-R数据故障；双模列尾机车台未配置GSM-R模块时不进行GSM-R数据测试；  
b) 双模列尾机车台配置了 $4 0 0 ~ \mathrm { M H z }$ 数字通信模块时，向出入库检测台发送 $4 0 0 ~ \mathrm { M H z }$ 数据测试命令，收到出入库检测台应答后判为 $4 0 0 ~ \mathrm { M H z }$ 数字数据正常， $1 0 \mathrm { ~ s ~ }$ 内未收到出库检测台应答

进重发，重发2次后仍未收到出入库检测台应答判为 $4 0 0 ~ \mathrm { M H z }$ 数字数据故障；双模列尾机车台未配置 $4 0 0 ~ \mathrm { M H z }$ 数字通信模块时不进行 $4 0 0 ~ \mathrm { M H z }$ 数字数据测试。出库检测台返回的数据测试应答含时间信息，双模列尾机车台收到后，如当前卫星定位信息效则动按出库检测台时间进校准。

# A.1.6 自检

双模列尾机车台动检，显示蓝按键提示信息和 $6 0 \ \mathrm { s }$ 倒计时，同时每隔5s发出一次语音提示“请按键试验”。按下“断连”“排风”“查询”“确认”等按键时，如果按键有效，对应的按键显示变为灰色，否则保持蓝显示；4个按键均有效或 $6 0 \ \mathrm { s }$ 倒计时结束时，双模列尾机车台停语提示，清除显示的按键提示和倒计时，显示自检和数据检测结果，30s后自动退回主界面（自检完成后，如数据测试尚未完成，等待过程中则显相应制式“数据测试进中”）。

# A.1.7 检测结果发送

双模列尾机车台同时以GSM-R和400MHz数字制式向出库检测台发送，出库检测台收到检测结果后以相应制式返回应答，检测结果发送应满如下应答重发机制要求：

a） 若10s内收到出库检测台应答，双模列尾机车台停重发，显示相应制式的“检测结果发送成功”；  
b) 若10s内未到出库检测台的应答，则双模列尾机车台自动重发检测结果，重发次数不超过2次；  
c) 若重发2次后10s仍发到出库检测台的应答，则双模列尾机车台显“检测结果发送失败”，退出检测结果发送大态

检测结果内容包括机车号、机车端号、空制号号及数量、模块配置状态、记录单元状态、校时前后时钟、本机IP地址、CIR卫星定位单元通信状、各式数据测试结果和RMS通信状态。

# A.1.8地面遥测

出库检测台遥测双模列尾机车台时，双模列尾机车台动进测试和检，但在检过程中不再进按键试验，检测结果中按键试验结果填无效字段。

# A.2 列尾机车台数据采集器

# A.3 列尾机车台数据分析系统

列尾机车台数据分析系统应满足下列要求：

b) 将读取的列尾机车台记录数据导人数据库，按日期、列尾主机ID、机车号和操作类别进行分类查询和统计；  
c) 打印数据分析报表。

# A.6.2 功能要求

模式以及 $\mathrm { G S M - R } + 4 0 0 \ \mathrm { M I }$ Iz双模模式下的作电流。

A.6.2.4 检测列尾主机排风功能。  
A.6.2.5 检测列尾主机排风量功能。  
A.6.2.6 检测列尾主机风压不正常报警功能。  
A.6.2.7 检测列尾主机电池欠压报警功能。
